# Doomlike

### Running

In game folder ```python Doomlike.py``` or ```python3 Doomlike.py```

\* pygame required


### Controls
**Menu:**

**```<UP>```** and **```<DOWN>```** to choose

**```<ENTER>```** to select

**```<ESC>```** to exit

**In game:**

**```<A>```** and **```<D>```** to move

**```<SPACE>```** to shoot

**```<ESC>```** to exit
